Accountability: About This Data Set

The data was collected with the help of Aware Framework The data primarily contains GPS sensor information over a week (October 09, 2018 to October 16, 2018) along with the timestamps. The table contains arrival timestamp, departure timestamp, address of the location, lat-long of the location and accuracy.

Where did this data come from?

The data was collected by capturing various sensor data which was performed with the help of the Aware Client on iOS running on Apple iPhone 6S which was always within a 10ft range from Shaleen.

Who has access to it?

Both read and write access to the dataset is restricted only to the data owner. Here we define data owner as the person whose data is collected, here Shaleen. However, the visualizations on this data and selected information is made available by this website to describe last week's story of Shaleen's life on this website.
